# Credit Risk Modeling

## Overview

This project predicts credit risk using machine learning techniques.

## Dataset

German Credit Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Scikit-Learn
* SMOTE
* SHAP

## Models Used

* Logistic Regression
* Random Forest

## Evaluation Metrics

* Accuracy
* Recall
* ROC-AUC

## Features

* Data Cleaning
* Exploratory Data Analysis
* Class Balancing using SMOTE
* Model Training and Evaluation
* SHAP Explainability

## Note

The original dataset did not contain a target risk column. A proxy Risk variable was created using Credit Amount and Duration for educational modeling purposes.
